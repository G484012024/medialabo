プロジェクト課題について

数当てゲーム

数当てゲームでは、乱数1~10の値を1つ決めて答えとし、ユーザは答えを予想して、最大3回まで入力できる。 ユーザの入力が外れた場合は、入力よりも答えが大きいか小さいかを表示。3回以内に答えを当てればユーザの勝ち そうでなければ負け。

演習の成果: 数当てゲーム

各授業回の学習事項

第1回 WWWと HTML

初めに、HTMLの大まかな使い方を学習した。 その後、それぞれ、HTMLテキストの要素」，「ハイパーリンク」，「画像の埋め込み」，その他の要素の紹介などの参考資料を参考に structure.hatmlとindex.htmlをVisual Studio Codeで作った。

演習の成果: 練習1-4 HTMLのさまざまな要素

第2回 CSS による Web ページの装飾

今回は、前回に引き続きHTMLについて学び、その中でもHTMLを装飾するときに記述するcssについて学習した。 cssは要素の見た目を変更する時に用いるもので、HTMLが先に与えられた後にcssを記述するという形になる。 index.htmlとproject.htmlのスタイルシートを作った。

演習の成果: 課題

第3回 JavaScript の基本

今回は、javaとjavascriptの違いを学び、javascriptをHTMlに追加する方法はscript要素を使って、javascriptを紐付けた。 また、コンソールのやり方を学んで、vscode上ではなくコンソール上でjavascriptを操作した。 JavaScripthはjavaと書き方が似ているように感じたが、javaとは違う言語なので、最初はほとんど意味がわからなかったが、 一つずつ、Javaと比べるとだんだんとわかった。 他にも、インラインコードやグローバル変数、ローカル変数、for-of構文について少し学んだ。 授業内課題としては、演習3-6,課題3-1(数当てゲームの作成その1),課題3-2(オブジェクトかレアデータを取り出す), 課題3-3(index.htmlの編集),課題3-4(リポジトリへの登録),課題3-5(オブジェクトからデータを取得する（プロジェクト課題)), 課題3-6(リポジトリへの登録)を行った。
